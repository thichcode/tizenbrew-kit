Samsung Tizen npm

shortvideo-tv

Facebook Reels feed player for Samsung Tizen 3+ TVs (2017+). Runs as an injected script in a TizenBrew web app — no native SDK needed.

Install

  1. Install TizenBrew on your Samsung TV
  2. Open TizenBrew on your TV
  3. Go to Module Manager (tab bar at the top)
  4. Add shortvideo-tv as a module

TizenTube (included by default) can be found on the home screen. Follow the same module manager flow to add ShortVideo TV.

Related projects

  • TizenBrew — modded web browser for Samsung Tizen TVs
  • TizenTube — ad-free YouTube client for Tizen (reference architecture)
  • TizenBrewInstaller — one-click installer for TizenBrew apps
  • TizenTubeCobalt — TizenTube port for Android and other non-Tizen devices

How it works

  • Runs entirely in the TV's WebView browser
  • Fetches a curated feed from the cloud resolver Worker
  • Resolves Facebook Reels via backend yt-dlp (or direct CDN playback)
  • 3-stage fallback: direct CDN → redirect → proxy stream
  • Designed for Samsung Tizen 3+ remote (D-pad, Back, Play/Pause)
